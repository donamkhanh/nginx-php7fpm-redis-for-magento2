# nginx-php7fpm-redis-for-magento2
A simple stack ready for Magento 2. Please note that redis term here is just only the php extension :)

### Packages:
* git
* vi
* zip/unzip
* php7.0
    * php7.0-fpm
    * php7.0-curl
    * php7.0-gd
    * php7.0-intl
    * php7.0-json
    * php7.0-pgsql
    * php7.0-phpdbg
    * php7.0-redis
    * php7.0-mysql
    * php7.0-cli
    * php7.0-xml
    * php7.0-mcrypt
    * php7.0-mbstring
    * php7.0-zip
* nginx
* composer

### Docker Compose
Please see [docker-compose.yml](./docker-compose.yml)
